:+1: worldcup-predict
===================

:boom: World Cup prediction with animals :smile:

:boom: Examples:


`$ worldcup-predict TEAM1 TEAM2`
```
  _______________
 | TEAM1 > TEAM2 |
  ---------------
   \
    \
        .--.
       |o_o |
       |:_/ |
      //   \ \
     (|     | )
    /'\_   _/`\
    \___)=(___/

```
